🚀 COMO RESTAURAR:
1. Copie os arquivos para /app/
2. Execute: yarn install (no frontend)
3. Execute: pip install -r requirements.txt (no backend)
4. Configure as variáveis de ambiente
5. Execute: supervisorctl restart all
